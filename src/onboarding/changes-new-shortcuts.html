Quick Folder Move

Apologies for another message on upgrade, but this one is important especially if you've been having trouble with the keyboard shortcuts. Unfortunately shortcuts are scarce in Thunderbird, and some I had picked were already taken. In some cases this caused the wrong action to be executed.

By default, I've now gone with just one shortcut assigned by default, allowing you to customize them to your needs instead. Note you can use the move shortcut and then the left/right arrow keys to adjust the action. Here are the currently assigned shortcuts, including any you have customized:

You can change these shortcuts or assign any that are unassigned in the shortcut manager, or temporarily go back to the legacy shortcuts in the add-on's preferences.

To make up for it, there are loads of new features. Many of them thanks to suggests from users like you!

  • A new compact layout, for those who prefer the style of the old version.
  • A new action to tag message with the labels in Thunderbird.
  • Pick a set of default folders to show instead of the recent folders.
  • Return of the option to exclude archive folders.
  • Don't show subfolders of trash, to avoid moving into a folder you've deleted.
  • The current account is being shown first within search results.
  • Search for multiple words/substrings to narrow down the search.
  • Fix for items automatically being selected when the mouse is over the popup.
  • Arrow key navigation wraps around.
  • Use Tab and Shift+Tab instead of the up/down arrow keys if you prefer.

If you have any questions, please reach out via the add-on's GitHub page. This is also where you can help improve future versions of the add-on.
