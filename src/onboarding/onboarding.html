Quick Folder Move

Welcome to Quick Folder Move! Getting to inbox zero can be difficult. There are many emails to triage, and it takes time to read and reply to each. Quick Folder Move is meant to reduce the time between emails. Those few extra clicks to move the message in a folder might not seem much, but with the amount of emails you receive every day it can add up.

Let's see how to make best use of this add-on!

Moving in context

On each message, there is a button that will allow you to move or copy emails to a folder of your choice. Once you are in the popup, you can use the following to navigate:

  • ⌘+Left/Right Arrow: Switch between 'move', 'copy' and other modes.
  • Ctrl+Left/Right Arrow: Switch between 'move', 'copy' and other modes.
  • Search box: Use this to search for part of the folder name. You might only need 1-2 letters!
  • Down/Up Arrow: Move between the search box and the folder items.
  • Enter: When you've picked the folder you'd like, press enter to move the message. If it is the first folder, press enter right away.
  • ⌘+Enter: Move the message, and directly go to the folder you moved the message to.
  • Ctrl+Enter: Move the message, and directly go to the folder you moved the message to.

The keys to boost your speed

Navigating with the keyboard is not confined to the popup. Ideally, you don't want to leave your keyboard when filing messages.

As available keyboard shortcuts in Thunderbird are scarce, we'll start out with just one shortcut to take action on email:

  • Control+Shift+M: Open the popup to move the selected email message.
  • Ctrl+Shift+N: Open the popup to move the selected email message.

In the add-ons manager, you can also assign additional shortcuts to open the popup directly with the copy, tag, or goto actions selected. Here, you can also customize the shortcut for moving messages. If you've used this add-on in the past please note Shift+M is no longer recommended due to restrictions in Thunderbird.
